# Sprocket Multiplayer

**Sprocket Multiplayer** is an open-source, work-in-progress multiplayer prototype mod for **Sprocket**.  
It allows players to connect and play together in a shared game environment.  

---

## Features

- Basic multiplayer support for Sprocket
- Experimental lobby system
- herobrine

> **This mod is still in development. Features are experimental and may be unstable.**
